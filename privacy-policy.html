Privacy Policy

Sleep Tracker · Last updated: March 28, 2026

1. Overview

Sleep Tracker is a personal-use application operated by a single individual (the developer). This policy explains what data is accessed and how it is handled.

2. Data Collected

The app accesses the following data from your Whoop account via the official Whoop API:

  • Sleep metrics (sleep onset, wake time, duration, disturbances, sleep performance score)
  • Recovery and nap data

The app also collects voice input entered manually by the user to fill remaining sleep tracking fields.

3. How Data Is Used

Data is used solely to populate the sleep tracking form within the app. It is not analyzed, aggregated, or used for any purpose beyond displaying it to the user within the same session.

4. Data Storage

No sleep or personal data is stored on any server. All data remains in the user's browser session and is cleared when the app is closed. No databases, analytics services, or third-party storage providers are used.

5. Data Sharing

No data is shared with any third party. The only external service contacted is the Whoop API to retrieve the user's own data, and the Anthropic API to process voice input — solely for the purpose of parsing sleep fields.

6. Authentication

The app uses Whoop's official OAuth 2.0 authentication. The developer does not have access to your Whoop password. Access tokens are stored only in the browser session and are not transmitted to any external server controlled by the developer.

7. User Rights

As the sole user of this application, you may revoke the app's access to your Whoop account at any time via your Whoop account settings under Connected Apps.

8. Contact

This app is for personal use only. For any questions, the developer can be reached via the GitHub repository at github.com/athyrvet/sleep-tracker.
